Este conjunto de documentos estabelece diretrizes e normas para governança de TI no setor público, integrando estruturas de conformidade baseadas em frameworks reconhecidos (CIS, NIST) e os instrumentos do PPSI. Os materiais incluem portarias oficiais que definem atribuições, procedimentos e fluxos de governança, bem como guias de implementação e ferramentas práticas (checklists, modelos de políticas e de avaliação de maturidade) para reforçar gerenciamento de riscos, controle de acesso, incidentes e conformidade regulatória.
